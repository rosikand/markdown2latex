Markdown2LaTeX

This is a Notable-flavored Markdown to LaTeX converter. Upload a .md file below and click the button to view your output. Then, save the output as a file.

Note: this was made by me for myself. When I write in markdown, I often use a set of standards customely developed in my mind and, as such, I wrote this program to tailor to those standards. Therefore, it is unlikely this will work for all inputs, but it should still produce a reasonable output.
